Repository: GeoPayLog
URL: https://github.com/crazyshrut/GeoPayLog
Language: JavaScript
Description: A frictionless expense tracker that answers "Where does my money go?" Built with React Native + Node.js. Zero login, just GPS-tagged spending insights.
Tech Stack: React Native (frontend), Node.js + Express (backend), MongoDB (database), Expo (development)
Purpose: Built as a mobile app to help users track daily expenses with automatic GPS location tagging. The idea was to remove friction - no sign-up needed, just open and start logging.
Tradeoffs: Chose React Native over native for faster development across platforms. Used MongoDB for flexible schema since expense categories can vary. Went with Expo for easier development but it limits some native module access. No authentication means data is device-local only.
Key Features: GPS-based expense tagging, city-wise spending summaries, visual charts for spending patterns

Repository: ImageEditor
URL: https://github.com/crazyshrut/ImageEditor
Language: Java
Description: A matrix-based command line image editor coded in Java on VS Code.
Tech Stack: Java (core), matrix operations for pixel manipulation
Purpose: Built as a learning project to understand image processing fundamentals using matrix operations. All image transformations (rotation, blur, sharpen, etc.) are done through direct pixel manipulation.
Tradeoffs: Command-line only - no GUI. Chose Java for strong typing and matrix handling. Processes images as 2D arrays which is memory-intensive for large images but simple to understand and implement.
Key Features: Image rotation, blur, sharpen, grayscale conversion, brightness adjustment - all through matrix math
